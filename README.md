Step to Create App:
1. install `inquirer`, `chalk` and `commander` 
2. create index.js and add `!#/usr/bin/env node` on the top of the file
3. open `package.json` file and add following script:
   ```
   "bin":{
      "ep":"./index.js"
   }
   ```



for Publish:
1. install `np` package with `npm install np`
